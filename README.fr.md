## ✨ Fonctionnalités

- 🔗 Intégration fluide entre SWADE, Stylish Action HUD et Better Rolls 2  
- ⚡ HUD automatique avec toutes les actions du personnage  
- 🎲 Entièrement intégré à Better Rolls 2  
- 🧼 Interface claire et organisée  
- 🚀 Plug and play (aucune configuration requise)  
- 🧩 Compatible avec Foundry VTT V13  

---

## 📦 Prérequis

- Système : SWADE (v5.1.0+)  
- Modules :
  - Stylish Action HUD  
  - Better Rolls for SWADE 2  

---

## 🚀 Installation

1. Ouvrez Foundry VTT  
2. Allez dans **Installer des modules**  
3. Cliquez sur **Installer un module**  
4. Collez l’URL :
https://raw.githubusercontent.com/CalielBR/swade-StylishActionHud-integration/main/module.json


---

## 🧠 Fonctionnement

Ce module connecte :
- Le système SWADE  
- Stylish Action HUD  
- Better Rolls 2  

👉 Résultat : un système de combat rapide, fluide et intuitif.
