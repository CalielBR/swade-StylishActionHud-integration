## ✨ Features / Funcionalidades

### 🇧🇷 Português
- 🔗 **Integração perfeita** entre SWADE, Stylish Action HUD e Better Rolls 2  
- ⚡ **HUD automático** com Atributos, Perícias, Armas, Poderes e Ações  
- 🎲 **Totalmente integrado ao Better Rolls 2**  
- 🧼 **Interface limpa e organizada**  
- 🚀 **Plug and play (sem configuração)**  
- 🧩 **Compatível com Foundry VTT V13**

### 🇬🇧 English
- 🔗 **Seamless integration** between SWADE, Stylish Action HUD, and Better Rolls 2  
- ⚡ **Automatic HUD population** with all character actions

- 🧼 **Clean and organized interface**  
- 🚀 **Plug and play (no configuration required)**

## 📦 Requirements

- **System**: [SWADE](https://foundryvtt.com/packages/swade) (v5.1.0+)  
- **Modules**:  
  - [Stylish Action HUD](https://github.com/glitchsmith/stylish-action-hud)  
  - [Better Rolls for SWADE 2](https://github.com/swade-foundry/swade-better-rolls)

---

## 🚀 Installation

1. Open **Foundry VTT**
2. Go to **Install Modules**
3. Click **Install Module**
4. Paste the Manifest URL:
https://raw.githubusercontent.com/CalielBR/swade-StylishActionHud-integration/main/module.json

---

## 🧠 How It Works

This module acts as a **bridge layer**, connecting:

- SWADE system data  
- Stylish Action HUD UI  
- Better Rolls 2 roll engine  

👉 Result: a unified, fast, and intuitive combat workflow.
