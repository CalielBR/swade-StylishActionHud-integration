## 🚀 Installation / Instalação / Installation

**PT:**  
1. Abra o **Foundry VTT**  
2. Vá em **Instalar Módulos**  
3. Clique em **Instalar Módulo**  
4. Cole a Manifest URL abaixo  

**EN:**  
1. Open **Foundry VTT**  
2. Go to **Install Modules**  
3. Click **Install Module**  
4. Paste the Manifest URL below  

**FR:**  
1. Ouvrez **Foundry VTT**  
2. Allez dans **Installer des modules**  
3. Cliquez sur **Installer un module**  
4. Collez l’URL du manifest ci-dessous  
https://raw.githubusercontent.com/CalielBR/swade-StylishActionHud-integration/main/module.json

## 🧠 How It Works / Como Funciona / Fonctionnement

- Dados do sistema SWADE
